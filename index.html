Bruno Coitinho

Desenvolvedor

Olá, me chamo Bruno Coitinho, sou nascido no Rio de Janeiro. Comecei minha jornada com tecnologia na adolescência quando fiz Curso Técnico de Informática. Atualmente estou me graduando em Sistemas de Computação e Matemática enquanto paralelamente estudo Desenvolvimento Web na EBAC.

Principais habilidades

HTML5

Tenho habilidades básicas sólidas em HTML, que é a linguagem fundamental para a construção de páginas web.

Além disso, compreendo a importância da semântica na criação de páginas web acessíveis e otimizadas para mecanismos de busca, aplicando tags apropriadas para destacar a estrutura lógica do conteúdo.

CSS3

Tenho habilidades sólidas em CSS para estilização de páginas web. Posso criar layouts complexos usando CSS Grid, layouts flexíveis com Flexbox e tornar páginas responsivas com Media Queries. Garanto compatibilidade entre navegadores e busco aprimorar minhas habilidades constantemente

Bootsrap

Tenho habilidades básicas em Bootstrap para o desenvolvimento de interfaces web responsivas. Consigo utilizar eficientemente o framework para criar layouts consistentes e componentes visualmente atraentes. Estou familiarizado com a utilização de classes, grade responsiva e componentes pré-construídos do Bootstrap para agilizar o processo de design e garantir a compatibilidade entre dispositivos.

JavaScript

Tenho habilidades básicas em JavaScript para tornar minhas páginas web dinâmicas. Posso manipular elementos do DOM, lidar com eventos e tenho noções básicas de jQuery para facilitar interações. Estou em busca de aprimorar minhas habilidades nessa linguagem.

Python

Estou em processo de aprendizado em Python, buscando construir uma base sólida na linguagem. Futuramente, pretendo direcionar meus estudos para aprender Django, visando a construção de aplicativos web robustos.

Git

Tenho habilidades básicas em Git para controle de versão, incluindo o uso de comandos essenciais como commit, push, pull e merge. Sou familiarizado com o GitHub, onde gerencio repositórios.

Compreendo os conceitos fundamentais de versionamento de código, contribuindo para a organização e rastreamento eficiente das mudanças em projetos colaborativos.
